# Копилка полезных ресурсов

Коллекция полезных ссылок для повседневного использования.

[Демо](https://levshirenin.github.io/link-saver/)

## Функции

- Категории и теги для удобной навигации
- Поиск по названию, описанию и URL
- Тёмная и светлая темы (по умолчанию — тёмная)
- Бейдж «VPN» для ресурсов, требующих VPN
- Адаптивный дизайн (мобильные и десктоп)

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

Статический экспорт генерируется в папку `out/`.

## Деплой

```bash
npm run deploy
```

## Структура проекта

```
├── pages/
│   ├── index.js          # Главная страница
│   ├── _app.js           # Обёртка с ThemeProvider
│   └── _document.js      # SSR-скрипт для темы
├── components/
│   ├── LinkList.js       # Фильтрация, поиск, группировка
│   ├── LinkItem.js       # Карточка ссылки
│   ├── ThemeContext.js    # Контекст темы
│   └── styles.js         # Общие стили
├── public/
│   └── links.json        # Данные ссылок
```
